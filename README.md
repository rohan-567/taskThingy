# TaskThingy

A task manager that's (hopefully) less miserable (currently only available for android)


<h2>Planned features:</h2>

- Visual time managment system
- Simple drag and drop capability
- Support for multi step routines
- Inbox for spur of the moment thought dumps
- Calendar integration
